HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata is information about data. It's stored in the <head> and is used by the Browser, Search Engines and other developers.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system with a purpose to make software development easier.
  • Version control helps to save snapshots or versions of a project, allowing a project to easily be restored to a past point if needed.
  • Other collaborators can create snapshots of their own to work on the project alongside you, but working on their own version (fork/branch) to ensure that they are not disturbing your version of the project.
  • Common Git Commands:
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a container/binding that stores items. Using the name of a variable you can easily refer to the data stored inside.
  • Javascript makes the webpage come to life through interactivity and dynamic behaviour!
